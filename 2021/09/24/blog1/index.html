n个球放入m个盒子问题

  1. 球相同，盒子不同，没有空盒子 n>=m

    插板法， C（n-1,m-1)

  2. 球相同，盒子不同，可以有空盒子

    假设每个盒子已经有一个球，插板法 C（n+m-1,m-1)

  3. 球不同，盒子相同，没有空盒子

    dp[n][m] = m * dp[n-1][m] + dp[n-1][m-1] 1 <= m < n

    第二类斯特林数

  4. 球不同，盒子相同，可以有空盒子

    第三种情况 枚举箱子的个数

  5. 球不同，盒子不同，没有空箱

    dp[n][m] * fac[m]

    dp[n][m] 指第三种情况， 因为球是不同的，盒子是相同的，给盒子定义顺序就好了

  6. 球不同，盒子不同，允许空箱，

    m的n次方

  7. 球相同，盒子相同，允许空箱。

    dp[n][m] = dp[n][m-1] + dp[n-m][m] n>=m

    dp[n][m] = dp[n][m-1] n < m

    dp[k][1] = 1, dp[1][k] = 1,dp[0][k] = 1;

  8. 球相同，盒子相同，没有空箱子

    dp[n-m][m],dp同第7种情况， n>=m

    0 n < m
